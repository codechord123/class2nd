# 1학기 프로그램 기능 차용 검토 (Phase 6 사전 탐색)

> 대상: `legacy/s1-index.html` (3,059줄) 전수 탐색 결과.
> 기준: 2학기에 이미 있는 것 / 차용 가치 있는 것 / 2학기 구조로 대체된 것.

## A. 차용 강추 — 2학기 취지에 잘 맞고 구현 비용 낮음

| # | 기능 | 1학기 모습 | 2학기 적용안 |
|---|---|---|---|
| 1 | 🐢 **독서 마라톤 진행 바** | 학급 목표 대비 진행률 트랙 위에서 거북이(🐢)가 결승선(🏁)으로 이동하는 애니메이션 위젯 | **짜파게티 파티 목표와 결합** — 홈·독서 탭 배너 아래에 "학급 전체 ○○○권 / 목표 ○○○권" 거북이 트랙. readingStats 1문서로 렌더 가능(추가 읽기 0) |
| 2 | ✍️ **감상문 필드 확장** | 제목·지은이·**출판사·줄거리·인상 깊은 장면·인용(quote)**·느낀점 | 2학기 폼은 제목/지은이/느낀점뿐. 1학기 감상문 열람 UI는 이미 전체 필드를 지원하므로 폼만 확장하면 됨 |
| 3 | 💾 **감상문 임시저장** | isDraft로 저장 → 이어서 작성, 정식 등록 시에만 권수 +1 | 초등학생이 긴 글을 한 번에 못 쓰는 현실 반영. 동일 방식 이식 |
| 4 | 🔢 **최소 글자수 설정** | 교사 설정에서 감상문 최소 글자수(기본 700자) 지정 | 교사 설정 탭에 항목 추가, 등록 시 검증 |
| 5 | 💬 **건의 댓글 + 대댓글** | 댓글·답글(@멘션 표시)·삭제, Shift+Enter 줄바꿈 | 건의글 문서 내 comments 배열로 이식(문서 1개 갱신, 읽기 증가 없음) |
| 6 | 📌 **공지 고정** | 건의글을 공지로 등록/내리기 → 상단 고정 | 게시판 상단 고정 + 홈 노출 옵션 |
| 7 | 🏪 **상품 카탈로그 + 보상 단계표** | 실버 보상표(10~100: 간식→문화상품권)·골드 보상표(10~100: 체육 연장→과자 파티), 상품 목록·가격 | 현재 자유 텍스트 신청 → 상품 선택형으로 개선. 교사가 카탈로그 편집. 골드 보상표는 학급 골드토큰(14개) 사용처로 연결 |
| 8 | 📝 **선생님 메모장** | 개요 대시보드에 메모(폰트 크기 조절) | 교사 탭 위젯으로 이식 (classData 1문서) |

## C. 대체 완료 — 차용 불필요 (2학기 구조가 상위 호환)

| 1학기 | 2학기 대체 |
|---|---|
| 1인1역 25개 역할 + 역할 통계 | Team 5역할 모둠 시스템 + 통계 |
| 일일점검(daily_records, 누가 점수 줬는지) | 일일 집계 + dailyScores/_meta |
| 명단/점수 대시보드(token_dashboard) | Team 통계 + 상점 잔액표 |
| 주간/월간/누적 점수 계산(클라이언트 전량 로드) | 서버식 사전집계(멱등) — **1학기 방식 재도입 금지** |
| mainState 단일 문서 구조 | 컬렉션 분리 — 재도입 금지 |

## 읽기 비용 관점 주의
차용하더라도 1학기의 구현 방식(전체 onSnapshot, mainState 몰아넣기)은 절대 따라가지 않는다. 위 적용안은 모두 "문서 1개 읽기" 또는 "기존 읽기에 무임승차" 방식으로 재설계한 것.
